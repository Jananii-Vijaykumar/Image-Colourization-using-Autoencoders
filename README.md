# 🎨 Image Colourization using Autoencoders

This project automatically restores colors to grayscale images using a Convolutional Autoencoder trained on the CIFAR-10 dataset.

---

## 📌 Features

- RGB to Grayscale conversion
- Convolutional Encoder
- Decoder Network
- Automatic image colorization
- Image comparison

---

## 🛠 Technologies

- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

---

## 📂 Workflow

```
Color Images
      │
      ▼
Convert to Grayscale
      │
      ▼
Encoder
      │
      ▼
Latent Features
      │
      ▼
Decoder
      │
      ▼
Colorized Image
```
